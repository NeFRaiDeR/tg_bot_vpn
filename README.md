# VPN Telegram Bot

Telegram бот для автоматизации продаж VPN подписок.

## 🚀 Возможности

- Регистрация и личный кабинет
- Пополнение баланса (вручную + платежная система)
- Продление подписки на 30/90/120/365 дней
- Бесплатный пробный период (7 дней)
- Дополнительные устройства (слоты)
- Автоматическая выдача конфигов из 3x-ui
- Админ-панель для управления пользователями
- Рассылки
- Поддержка пользователей

## 🛠 Технологии

- Python 3.10+
- Aiogram 3.x (Telegram Bot API)
- PostgreSQL + asyncpg
- 3x-ui API
- Docker

## 📦 Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте `.env` по образцу
4. Запустите: `python main.py`
